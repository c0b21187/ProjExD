## 迷路ゲーム：逃げろこうかとん（ex04/dodge_bomb.py）
### ゲーム概要
- ex04/dodge_bomb.pyを実行すると，1300x700のcanvasに墓場が描画され，おばけをよけるようにこうかとんを移動させるゲーム

### 操作方法
- 矢印キーでこうかとんを上下左右に移動する
### 追加機能
- 爆弾（おばけ）の追加
- 爆弾をおばけに変更
- 背景を変えた
